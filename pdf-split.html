How to Split a PDF

Drop a PDF file onto the converter. Thumbnails of all pages will appear. Choose a split mode: "All pages" creates one PDF per page, "Custom page ranges" lets you specify sections like 1-3,5,7-9, and "Every N pages" splits into chunks of N consecutive pages. Click "Split PDF" and download individual files or a ZIP archive.

Each split operation creates separate PDF documents that preserve the original formatting, fonts, and layout. When saving individual pages, each PDF is a fully valid document that can be opened in any PDF reader.

Understanding Page Ranges

Page ranges use the format: 1-3,5,7-9. This creates PDFs for pages 1 through 3, a separate PDF for page 5, and another for pages 7 through 9. You can mix single pages and ranges. Page numbering starts at 1.

Split PDF vs Keeping Intact

Pros of Splitting: Easier to share specific sections, smaller individual files for email, organize content by topic, extract only needed pages.

Cons of Splitting: Loses continuous page numbering, separate files to manage, no single-document search across all pages.

Single Pages vs Page Ranges

Single Pages: Maximum granularity â€” each page is independent. Best for distributing individual forms, invoices, or certificates.

Page Ranges: Preserves logical groupings like chapters or sections. Each output file remains a multi-page document with context.

Batch Download Benefits

ZIP Archive: Download all split files in one click. The ZIP preserves page order with numbered filenames like document-page-1.pdf, document-page-2.pdf.

Individual Downloads: Click any split file to download it separately. Useful when you only need one or two specific pages from the set.
