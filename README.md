# ConvertPivot

**40+ free online converter tools — 100% browser-based, zero uploads.**

Convert files, data, and formats privately right in your browser. No signup, no server uploads, no limits.

## Tools

- **Developer**: Binary Converter, Hex to RGB, Timestamp Converter, JSON Formatter, Base64 Encoder/Decoder, Password Generator, Word Counter
- **Contacts & CRM**: VCF to CSV, CSV to VCF, VCF to XLSX
- **Financial**: OFX/QFX/QBO to CSV, CSV to QBO/OFX/QFX, PDF Bank Statement to CSV, Per-bank converters (Chase, Wells Fargo, BofA)
- **Notebooks**: IPYNB to PDF/DOCX/HTML/PY
- **Media**: HEIC to JPG/PDF, AVIF to JPG, WebP to JPG, SVG to PNG, and reverse converters
- **GPS & Fitness**: GPX to KML/CSV, FIT to GPX/CSV, CSV to GPX, KML to GPX

## Architecture

Every conversion runs client-side using vanilla JavaScript and WebAssembly. Files never leave the device. Built with vanilla HTML/CSS/JS — no build step, no NPM, no backend.

## Deployment

Deployed on Vercel with `cleanUrls: true`.
