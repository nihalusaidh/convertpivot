How to Convert EPUB to PDF

July 13, 2026

EPUB is great for reading on phones and e-readers because text reflows to fit your screen. But sometimes you need something fixed — maybe you want to print a chapter, share a section with someone who does not have an e-reader, or send a PDF to a publisher.

Converting EPUB to PDF gives you a print-ready document with page numbers and fixed formatting. Fonts, margins, page breaks — everything stays exactly as you set it. Here is how to do it for free.

Method 1: Using Calibre (Desktop Application)

Calibre is the most powerful free e-book management application available. It supports every major e-book format, provides extensive conversion options, and can manage libraries of thousands of books. Calibre runs on Windows, macOS, and Linux.

Steps: Download and install Calibre from calibre-ebook.com. Open Calibre and click Add Books to import your EPUB file. Select the imported book in the library, then click Convert Books > Convert Individually. In the conversion dialog, set the output format to PDF in the top-right corner. Configure output options including page size (letter, A4, or custom), margins, default font size, and header/footer settings. Click OK to start conversion.

Calibre offers fine-grained control over the PDF output. You can set the paper size, font family and size, line spacing, margins, justification, headers and footers with page numbers, table of contents generation, and metadata inclusion (title, author, cover image). For bulk conversions, Calibre supports batch processing and command-line automation.

Method 2: Using Online Converters

For quick, single-book conversions without installing software, online converters are the most convenient option. A simple web search for "EPUB to PDF converter" returns dozens of free services. The best online converters offer privacy-oriented processing where your files are not stored on servers after conversion.

To use an online converter: navigate to the converter website, upload your EPUB file (most accept files up to 50-100 MB for free), select PDF as the output format, optionally configure page size and margin settings, and click convert. After a few seconds to a minute, download the resulting PDF file.

Online converters typically delete uploaded files from their servers within a few hours for privacy. For sensitive documents, consider using a desktop application like Calibre instead, since the file never leaves your computer. Always review the privacy policy of any online converter before uploading copyrighted or personal content.

Method 3: Using Adobe Digital Editions

Adobe Digital Editions is a free e-book reader that can display EPUB files and, with some configuration, convert them to PDF. It is particularly useful for EPUB files with Adobe DRM, which other tools cannot open.

Steps: Open the EPUB file in Adobe Digital Editions. Go to File > Print. In the Print dialog, select "Adobe PDF" as the printer (or "Microsoft Print to PDF" on Windows, "Save as PDF" on Mac). Configure page settings — letter or A4 size, orientation, and margins. Click Print or Save to create the PDF version.

This method works for any file that Adobe Digital Editions can display, including DRM-protected EPUBs that you have authorized with your Adobe ID. The print-to-PDF approach works for individual books but is impractical for batch conversions. The output quality depends on the page settings — use high-quality print settings for the best results.

Method 4: Using Command-Line Tools (Pandoc / ebook-convert)

For users comfortable with the command line, Pandoc and Calibre's ebook-convert tool offer powerful, scriptable EPUB to PDF conversion. These tools are particularly useful for automated workflows, batch processing, and integration into publishing pipelines.

Using Calibre's ebook-convert: ebook-convert input.epub output.pdf --paper-size a4 --margin-top 20 --margin-bottom 20 --margin-left 20 --margin-right 20. This command converts the EPUB to a PDF with A4 page size and 20-point margins. You can specify dozens of options for complete control over the output.

Using Pandoc: pandoc input.epub -o output.pdf --pdf-engine=xelatex -V geometry:margin=1in. Pandoc requires a LaTeX distribution installed (like MiKTeX on Windows or TeX Live on macOS/Linux). While more complex to set up, Pandoc produces exceptionally clean PDF output with professional typography.

Tips for Better EPUB to PDF Conversion

The default page size in most converters is US Letter (8.5x11 inches) or A4 (210x297 mm). For fiction books, consider using a smaller page size like 5x8 inches or 6x9 inches for a more traditional book feel. For technical documents, letter or A4 with two-column layout can be more readable.

Font choices matter for readability in PDF output. Serif fonts like Garamond or Book Antiqua are traditional for book text. Sans-serif fonts like Helvetica or Arial work well for technical documents. Most converters let you choose embedded fonts or use system fonts. For maximum portability, embed common fonts in the PDF.

Check that the table of contents transfers correctly. EPUB files typically contain navigation metadata that can be converted to PDF bookmarks. Enable the table of contents/outline option in your converter for easy PDF navigation. Also verify that images, tables, and special formatting render as expected in the PDF output.

Frequently Asked Questions

  1. What is the difference between EPUB and PDF? EPUB reflows text to fit your screen — great for phones and e-readers. PDF locks the layout — great for printing and sharing fixed-format documents.
  2. Can I convert EPUB to PDF for free? Yes. Calibre is free and powerful. Online converters also work for occasional use. No payment needed.
  3. Will my formatting be preserved? Mostly, but expect some differences. EPUB is fluid, PDF is fixed. Font sizes, margins, and line spacing may shift a bit.
  4. Can I convert DRM-protected EPUBs? Not with standard tools. Removing DRM first may violate copyright. Only convert files you own that are DRM-free.
  5. What is the best EPUB to PDF converter? Calibre for desktop — it handles batch conversion and gives you fine control. Online converters for quick one-off jobs. I use Calibre for my library and an online tool for single books.
