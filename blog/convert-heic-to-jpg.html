How to Convert HEIC to JPG on Windows, Mac, and Online

July 13, 2026

If you own an iPhone or iPad, your photos are saved in HEIC format by default. HEIC (High Efficiency Image Container) is Apple's modern image format introduced in iOS 11. It uses HEVC compression to create files roughly half the size of JPEG while maintaining similar quality. The problem? HEIC isn't universally supported. Many Windows applications, email clients, and websites can't open these files. This guide walks through three reliable methods to convert HEIC to JPG so your photos work everywhere.

What Is HEIC and Why Convert It?

HEIC is Apple's default image format adopted since iOS 11 and macOS High Sierra. It wraps HEVC-encoded image data in a container that can also store image sequences, depth maps, and metadata. The main advantage is storage efficiency — a 5 MB JPEG can become a 2.5 MB HEIC with no visible quality loss. However, compatibility is limited. Windows 10 and 11 need a paid codec extension. Many online services reject HEIC uploads. Email attachments may not render. Converting to JPG solves these problems while keeping your photos accessible on any device.

Method 2: Convert HEIC to JPG on Mac Using Preview

If you're on a Mac, the built-in Preview app can convert HEIC files without additional software. This is ideal for one-off conversions or when you're working offline.

  1. Open the HEIC file in Preview (it's the default image viewer on macOS).
  2. Go to File > Export. Alternatively, use File > Export Selected Images if you have multiple files open.
  3. Choose JPEG as the format. Adjust the quality slider — 80-90% offers a good balance between file size and quality.
  4. Save. The new JPG file appears alongside your original HEIC.

For batch conversion, select all the HEIC files you want to convert in the Preview sidebar, then go to File > Export Selected Images. Choose JPEG as the format and click Export. Preview will convert every file in one operation.

Method 3: Convert HEIC to JPG on Windows

Windows doesn't natively support HEIC because HEVC (H.265) encoding requires a licensing fee from Microsoft. However, you have two options to work around this:

Option A: Install the HEIF Image Extensions. Microsoft offers a paid extension ($0.99) from the Microsoft Store. Once installed, Windows Photos can open HEIC files, and you can export them to JPG by right-clicking and selecting "Save as" or using Photos' File > Save a Copy. For batch conversion, select multiple HEIC files, open them in Photos, and export each one.

Option B: Rename and use an online converter. Since the HEIF extension costs money, many Windows users prefer using a free online converter like ConvertPivot. Simply drag your HEIC files into the browser and download the JPG outputs — no payments, no extensions, no uploads to a server.

Frequently Asked Questions

Why can't I open HEIC on Windows?
Windows lacks native HEIC support because the HEVC (H.265) codec requires licensing fees. You need the HEIF Image Extensions from the Microsoft Store or convert to a universal format like JPG.

Is HEIC better than JPEG?
HEIC offers roughly 50% better compression than JPEG at the same quality level. For storage-constrained devices like iPhones, this is a major advantage. However, JPEG wins on compatibility — it is supported by every device, browser, and application in existence.

Does WhatsApp convert HEIC?
Yes. When you share a HEIC photo via WhatsApp, the app automatically converts it to JPEG before sending. Recipients on Android, Windows, or older iPhones will receive a standard JPG that they can open without issues.

How to batch convert HEIC to JPG?
For batch conversion, use ConvertPivot's online converter which handles unlimited files at once. On Mac, use Preview's File > Export Selected Images with multiple files selected. On Windows, batch conversion is cumbersome without third-party software, making online tools the best option.

Does HEIC lose quality when converted to JPG?
Some quality loss is theoretically possible since HEIC retains more detail at the same file size. However, at high JPEG quality settings (90%+), the difference is visually imperceptible for photos viewed on screens, shared on social media, or printed at standard sizes.
