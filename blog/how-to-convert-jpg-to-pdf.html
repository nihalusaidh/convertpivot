How to Convert JPG to PDF in Browser

July 13, 2026

I convert JPG images to PDF all the time — for printing, combining scanned pages into a single document, or sending a portfolio to a client. Doing it in your browser is fast, free, and needs no software.

PDF keeps everything looking the same on any device. Unlike a JPG that someone might accidentally crop or resize, a PDF is locked. Here are the best ways to do it.

Method 2: Using the Browser's Built-in Print Feature

Every modern web browser includes a Print to PDF feature that can convert JPG images to PDF without any external tools. This method is built into Chrome, Firefox, Edge, and Safari and requires no online service.

Steps: Open a new browser tab. Drag and drop your JPG file onto the tab, or use Ctrl+O (Cmd+O on Mac) to open the image. Once the image is displayed full-size in the browser, press Ctrl+P (Cmd+P on Mac) to open the Print dialog. In the Destination or Printer section, select "Save as PDF" or "Microsoft Print to PDF." Configure page settings (paper size, margins, orientation) and click Save. Choose a location for your PDF file.

This method is limited to one image per PDF. For multiple images, you would need to repeat the process for each image and then use a PDF merger tool to combine them. It is best suited for quick single-image conversions when you do not need advanced options like compression adjustment or image arrangement.

Method 3: Using Microsoft Print to PDF (Windows)

Windows includes a built-in "Microsoft Print to PDF" feature that functions as a virtual printer. Any application that can print — including image viewers, photo editors, and file explorers — can use this feature to create PDF files from JPG images.

Steps: Right-click your JPG file in File Explorer and select Open with > Photos (or any image viewer). Click the Print button (or press Ctrl+P). In the Print Pictures dialog, select "Microsoft Print to PDF" as the printer. Configure layout options (full-page print, contact sheet, or wallet-size). Choose paper size and quality settings. Click Print, then select a destination folder and filename for the PDF.

Windows also supports selecting multiple JPG files in File Explorer, right-clicking, and selecting Print. This opens the Print Pictures dialog with all selected images, allowing you to print them all to a single PDF — each image becomes a separate page. This is the fastest method for Windows users to combine multiple JPGs into one PDF without third-party software.

Method 4: Using macOS Preview

Mac users have a powerful built-in tool for JPG to PDF conversion: the Preview application. Preview can convert single images or combine multiple images into a single PDF document with full control over page order and settings.

Steps for single image: Open the JPG in Preview. Click File > Export as PDF. Configure the filename and location, then click Save. For multiple images: Select all JPG files in Finder and open them together in Preview. The images appear in the sidebar thumbnails. Arrange them by dragging thumbnails into the desired order. Select all thumbnails (Cmd+A) and click File > Print. In the Print dialog, click PDF > Save as PDF. All selected images will be combined into a single multi-page PDF.

Preview also offers more advanced PDF features like adding annotations, adjusting color, resizing images, and setting passwords. For Mac users who frequently convert images to PDF, Preview's built-in capabilities rival dedicated PDF software.

Tips for Better JPG to PDF Conversion

Image resolution matters for the final PDF quality. For documents that will be viewed on screen, 150 DPI is sufficient. For printing, use 300 DPI or higher. Most JPG files from modern cameras and smartphones are high enough resolution for excellent PDF output.

Consider the page size of your output PDF. For standard documents, use letter (8.5 x 11 inches) or A4 (210 x 297 mm). For photos, auto-fit or custom sizes preserve the original aspect ratio. For presentations, landscape orientation with 16:9 or 4:3 aspect ratios work best.

File size is another consideration. High-resolution JPG images can produce large PDF files. Most converters offer compression options. For email or online sharing, moderate compression (balanced quality/size) is recommended. For archival or print quality, use minimal compression.
