Frequently Asked Questions

Morse code is a method of encoding text using sequences of dots (.) and dashes (-). It was developed in the 1830s by Samuel Morse and Alfred Vail for telegraph communication. Each letter and number has a unique sequence.
In Morse code, dots (.) represent short signals and dashes (-) represent long signals. A space between parts of the same letter is one unit, between letters is three units, and between words is seven units.
