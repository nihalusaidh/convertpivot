Frequently Asked Questions

A bit is the smallest unit of data (0 or 1). A byte is 8 bits. Bytes are used for file sizes, while bits are used for data transfer speeds (e.g., 100 Mbps = 12.5 MB/s).
Hard drive manufacturers use SI units (1 GB = 1,000,000,000 bytes) while operating systems use binary units (1 GiB = 1,073,741,824 bytes). A 1 TB drive = 1,000,000,000,000 bytes = ~931 GiB in Windows.
MB (Megabyte) uses SI decimal: 1 MB = 1,000 KB = 1,000,000 bytes. MiB (Mebibyte) uses binary: 1 MiB = 1,024 KiB = 1,048,576 bytes. MiB is used by most operating systems for file sizes.
In SI decimal: 1,000 MB = 1 GB. In binary: 1,024 MiB = 1 GiB. Always check which system you're using.
